# 飞机大战设计实现

基于 Unity 引擎开发的 3D 飞机大战游戏。

## 游戏介绍

玩家控制战机在太空中飞行，通过鼠标点击发射子弹击落不断下落的小行星。游戏包含生命值、得分和目标分数系统，击落小行星可获得分数，被小行星撞击会损失生命值。

## 场景说明

- **StartScene** - 开始界面，任意键进入游戏
- **GameScene** - 主游戏场景
- **WinScene** - 胜利界面，达成目标分数后进入
- **LoseScene** - 失败界面，生命值归零后进入

## 操作方式

- **鼠标左键** - 发射子弹
- **鼠标移动** - 控制战机左右移动
- **任意键** - 开始游戏 / 返回主菜单

## 核心功能模块

| 脚本 | 功能 |
|------|------|
| Begin | 开始界面，检测任意键进入游戏场景 |
| Move | 玩家战机移动控制，限定屏幕边界内 |
| Shoot | 鼠标点击发射子弹 |
| Bullet | 子弹移动及碰撞销毁 |
| Hp | 玩家生命值管理，受伤闪烁无敌、音效、碰撞检测 |
| Attack | 小行星被子弹击中后的销毁与爆炸特效 |
| Stone | 小行星下落与旋转运动 |
| Random_Stones | 随机生成小行星 |
| Score | 全局分数管理，单例模式，目标分数判定 |
| GameUIManager | 游戏 HUD 显示（生命值、得分、目标） |
| ScrollingBackground | 滚动星空背景 |
| BacktoStart | 结束界面，任意键返回主菜单 |

## 技术要点

- Unity 引擎，C# 脚本开发
- 使用 Particle System 实现爆炸特效
- 对象池思想（动态生成与销毁游戏对象）
- 单例模式管理全局分数
- 碰撞检测与触发器机制
- 无敌帧与闪烁效果提升游戏体验

## 开发环境

- Unity 2022.3 LTS
- C#
